Left click (and/or hold down): Creates a blob on the mesh and moves it around
Right click: deposits the blob on the mesh (cant be reselected)
w, s: Controls the height of the blob
a, d: Controls the width of the blob
F1: Changes mouse controls
F2: Changes shading effect
Up Arrow: zooms into mesh
Down Arrow: zooms out from mesh
Mouse Movement: moves the camera around the mesh when in camera mode

File uses a command called system('cls'); to clear the screen. Im not sure if it works on anything other than windows. Its in main.cpp located where the handle for the keyboard is.
